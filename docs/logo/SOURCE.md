# Logo Source Notes

The current splash logo is intentionally clean and text-based.

- App runtime implementation: `CleanHomeLogo()` in `app/src/main/java/com/coloursprout/kids/MainActivity.kt`
- Editable SVG concept source: `docs/logo/colour_my_world_logo.svg`
- Date updated: 2026-07-11

The logo uses simple text, a white rounded panel, and a small set of colour accents. The previous paint-splash vector was removed from the app because it looked busy and could appear clipped on phone screens.
