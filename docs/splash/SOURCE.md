# Splash Background Source Note

The shipped splash/home background is a generated clean SVG-style vector.

- Android runtime vector: `app/src/main/res/drawable/splash_art_room.xml`
- Editable SVG concept source: `docs/splash/clean_kids_home_background.svg`
- Date integrated: 2026-07-11

The background is original for this project. It uses pastel bands, simple clouds, soft rainbow arcs, and a few small colour accents so the home screen feels kid-friendly without visual clutter.
