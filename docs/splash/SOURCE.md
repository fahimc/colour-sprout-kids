# Splash Background Source Note

The shipped splash background is an Android vector drawable at `app/src/main/res/drawable/splash_art_room.xml`.

The visual direction was chosen after searching for colourful, child-safe public-domain SVG backgrounds. The closest public-domain reference found was:

- Title: Vector image of sunny living room scene
- Site: FreeSVG / PublicDomainVectors
- URL: https://freesvg.org/vector-image-of-sunny-living-room-scene
- License shown on source page: Public Domain
- SVG publisher shown on source page: OpenClipart
- Local reference preview: `docs/splash/sunny_living_room_reference.png`

The final shipped vector was adapted into an original art-room splash so no Disney, Pixar, Marvel, Star Wars, Pokemon, Peppa Pig, Paw Patrol, or other branded/copyrighted character artwork is used.
