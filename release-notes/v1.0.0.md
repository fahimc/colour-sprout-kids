# ColourSprout Kids v1.0.0

Initial Android release.

- Jetpack Compose kid-friendly colouring app.
- 32 generated CC0-style starter pages with masks, thumbnails, metadata, and license manifest.
- Tap-to-fill with cached mask regions.
- Brush, crayon, marker, glitter, eraser, eyedropper, undo, redo, clear, save, and PNG export.
- Offline-first app with no ads, analytics, accounts, or tracking.
