# Colour My World v1.0.5

Usability and colouring correctness fix.

- Removed the accidental green full-canvas selection overlay.
- Prevented fill and brush tools from colouring mask regions that touch the image border, which stops background/open paper fills.
- Replaced tiny circular tool buttons with readable game-style rounded buttons.
- Kept Save visible in portrait instead of hiding it at the end of a scroll strip.
- Renamed action buttons to readable labels like UNDO, REDO, AREA, ALL, and SAVE.
